# Security Policy

## Supported Versions
We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
ISOSTASY is critical infrastructure. We take security seriously.

**Do not open public issues for security vulnerabilities.**

Please report sensitive discoveries via our Responsible Disclosure process:
1.  **Email:** Send a detailed report to `security@isostasy.org` (or your preferred secure channel).
2.  **Encryption:** Use our PGP key [Insert Key Link] if possible.
3.  **Timeline:** We acknowledge receipt within 48 hours and aim to provide a resolution or public advisory within 30 days.

## Scope
This policy covers the core engine and official plugins. It does not cover third-party forks or external data sets used with the tool.
